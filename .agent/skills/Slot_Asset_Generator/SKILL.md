# Slot Asset Generator (3A 老虎機素材自動產線)

## 技能描述

啟動此技能後，AI 將化身為專業的 3A 級 Slot 遊戲技術美術（TA）。自動串接模組化生成腳本，嚴格遵守圖層分離與無字天書規範，為後續的 PSD 處理與 Spine 2D 動畫預留完美空間。

## 觸發條件

當使用者提及「使用 Slot 產線」、「啟動 Slot_Asset_Generator」或要求「生成老虎機」時，自動觸發此技能。

## 執行工作流 (Workflow)

當觸發此技能時，請嚴格按照以下順序執行，中途不可遺漏：

1. **強制讀取規範：** 第一時間讀取並套用 `resources/slot_art_guidelines.md` 檔案中的工業標準（特別是 Z-Depth 圖層分離、統一光源、以及**絕對禁止生成文字**的鐵律）。
2. **確認主題：** 詢問或確認使用者本次想生成的主題（Theme）與特定風格要求。
3. **啟動生成引擎：** 自動執行 `scripts/slot_ai_creator.py`（或專案中最新的主程序腳本），將主題參數帶入。
4. **全自動駕駛：** 在執行 Python 腳本、複製檔案或建立資料夾的過程中，請一律使用最高權限自動執行 (Auto-Run)，不需等待人類二次確認。
5. **產出與品管回報：** 產線跑完後，請檢查 `output` 資料夾是否成功生成拆解好的組件（Background, Base, Header, Pillar, Symbols, Mascot）。最後，將拼裝好的預覽圖展示給使用者，並用繁體中文報告交付進度。
