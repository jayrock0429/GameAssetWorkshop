# 📌 Pinterest Top 6 Trending Slot UI Styles (AI Style Guide)

這份文件定義了在 Pinterest 上最受歡迎的 6 大老虎機 (Slot Game) UI 與美術風格。
當使用者選擇以下特定風格時，AI 產圖引擎必須嚴格遵守該風格對應的色彩學、材質學與發光邏輯。

## 1. Cyberpunk / Neon Sci-Fi (電馭叛客/霓虹科幻)

- **視覺基調：** 充滿未來感、高科技冷酷與強烈霓虹色彩對比。
- **色彩學：** 深色背景 (黑、深紫、暗藍) 搭配極端高亮度的霓虹光 (螢光粉紅、青藍色 Cyan、螢光綠、黃色)。
- **材質學：** 格柵狀的金屬網、高光澤深色金屬、碳纖維、霧面玻璃 (Frosted Glass)、數位 LED 顯示器、雷射光束。
- **UI 特徵：** 俐落的幾何線條、全像投影效果 (Hologram)、動態掃描線、充滿科技感的細節標籤。

## 2. Oriental Wealth / Asian Theme (東方財富/亞洲風格)

- **視覺基調：** 喜氣、華貴、傳統與富麗堂皇的發財主題，深受亞洲市場喜愛。
- **色彩學：** 大量的正紅色、財富金 (Rich Gold)、翡翠綠、喜慶紫。
- **材質學：** 拋光黃金 (Polished Gold) 與耀眼的鏡面反射、細緻的玉石與翡翠質感、絲綢(Silk)、傳統漆器、朱砂。
- **UI 特徵：** 雲紋、龍、元寶、銅錢等傳統圖騰轉化為 3D 浮雕，UI 邊框必須帶有極具份量感的厚重金屬邊緣。

## 3. Ancient Egypt / Mythology (古埃及/神話)

- **視覺基調：** 神秘、歷史悠久、神聖而莊嚴。
- **色彩學：** 沙漠金、土黃色、青金石藍 (Lapis Lazuli)、曜石黑。
- **材質學：** 風化的神廟石材、粗糙的砂岩、刻有象形文字 (Hieroglyphics) 的石板、斑駁但閃耀的古金屬。
- **UI 特徵：** 背景通常配有神廟列柱或火炬，UI 畫框像是從古老化石碑中雕刻而出，帶有神秘的魔法能量微光。

## 4. Sweet & Candy / Casual Cute (甜美糖果/休閒可愛)

- **視覺基調：** 輕鬆、愉悅、可口、適合輕度休閒玩家。
- **色彩學：** 明亮且高明度 (High Value) 的馬卡龍色調 (粉紅、粉藍、鵝黃、薄荷綠)，幾乎不用純黑色。
- **材質學：** 晶瑩剔透的軟糖 (Gummy/Jelly)、反光的果凍、帶有糖霜的餅乾、蓬鬆的棉花糖、圓潤無死角的塑膠材質。
- **UI 特徵：** 所有邊角必須是圓弧形 (Rounded Corners)、絕對沒有尖銳銳角。UI ボタン要看起來「非常 Q 彈可口」。

## 5. Classic Vegas / Retro Casino (經典維加斯/復古賭城)

- **視覺基調：** 懷舊、經典、五光十色的拉斯維加斯實體機台感。
- **色彩學：** 經典的賭場紅黃綠配色、復古招牌的溫暖燈泡光 (暖黃、橘)。
- **材質學：** 閃爍的跑馬燈泡 (Marquee Lights)、紅絲絨 (Red Velvet)、亮面塑膠機台外殼、黃銅與閃耀的鉻金屬 (Chrome)。
- **UI 特徵：** 像是實體物理按鈕 (Mechanical Buttons)، 帶有立體的金屬按壓厚度，以及非常誇張且閃爍的 Jackpot 招牌。

## 6. Fantasy Adventure / Magic (奇幻冒險/魔法)

- **視覺基調：** 史詩、魔法、RPG 冒險。
- **色彩學：** 深邃的森林綠、魔法紫、神聖金、幽暗藍。
- **材質學：** 魔法水晶 (Glowing Crystals)、古老且帶有木紋的魔法書皮、發光的盧恩符文 (Runes)、神秘的金屬鍛造兵器。
- **UI 特徵：** UI 介面常被藤蔓纏繞、或鑲嵌著發光的魔法寶石。按鈕看起來像是神秘的魔法陣或附魔石板。
